# IRS System

An end-to-end IoT project using ESP32, RSSI, ThingSpeak, and Streamlit.

## Components
- ESP32 Transmitter
- ESP32 Receiver
- Streamlit Dashboard

## Folder Structure
ui/
├── app.py
├── Transmitter_ESP32/
├── Receiver_ESP32/

## Description
The transmitter ESP32 measures RSSI and distance and uploads data to ThingSpeak.
The receiver ESP32 fetches cloud data and performs hardware actions.
The Streamlit dashboard visualizes the data.
